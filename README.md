# lazyprj

Go製のターミナルUI（TUI）で、tmuxセッションを対話的に管理するツールです。  
[Bubble Tea](https://github.com/charmbracelet/bubbletea) + [lipgloss](https://github.com/charmbracelet/lipgloss) で実装されています。

---

## 機能

- プロジェクト一覧の表示（起動中・停止中をリアルタイム表示）
- tmuxセッションの起動 / 停止 / 再起動 / アタッチ
- **自動起動**: tmuxセッションがゼロの状態で起動すると `★` のプロジェクトを一括起動し、アタッチ先をTUIで選択できる
- ウィンドウ・ペイン設定の対話編集（エディタ画面）
- スキャンディレクトリから未登録プロジェクトを一覧表示して追加（スキャン画面）
- スキャン結果のプロジェクトを **スキップ済み** に設定し、次回以降のスキャンで非表示にできる
- 設定は `~/.config/lazyprj/config.json` に保存

---

## インストール

```bash
go install github.com/masaway/lazyprj@latest
```

またはソースからビルド：

```bash
git clone https://github.com/masaway/lazyprj
cd lazyprj
go build -o lazyprj .
# PATHの通った場所に移動（例）
mv lazyprj ~/.local/bin/
```

### 依存関係

- Go 1.24.2 以上
- tmux

## 設定ファイル

設定ファイルは初回起動時に自動作成されます。

| パス | 説明 |
|------|------|
| `~/.config/lazyprj/config.json` | 設定ファイル |

`XDG_CONFIG_HOME` が設定されている場合は `$XDG_CONFIG_HOME/lazyprj/config.json` を使用します。

### プロジェクト設定の主なフィールド

```json
{
  "projects": [
    {
      "name": "myapp",
      "path": "~/work/myapp",
      "auto_start": true,
      "description": "メインの開発プロジェクト",
      "windows": [
        {
          "name": "dev",
          "layout": "even-horizontal",
          "panes": [
            { "dir": ".", "command": "", "execute": false },
            { "dir": ".", "command": "npm run dev", "execute": true }
          ]
        }
      ]
    }
  ],
  "skipped_paths": [
    "/home/user/work/old-project"
  ],
  "settings": {
    "scan_directory": "~/work"
  }
}
```

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `name` | string | tmuxセッション名 |
| `path` | string | プロジェクトのルートディレクトリ |
| `auto_start` | bool | 起動時に自動起動するか（省略時: false） |
| `description` | string | 説明文（省略可） |
| `windows` | array | ウィンドウ・ペイン構成 |

### スキップ済みパス

`skipped_paths` にはスキャン画面で `x` キーによりスキップしたプロジェクトのフルパスが保存されます。  
一覧に表示したい場合は該当パスを削除するか、スキャン画面のスキップ済みセクションから `x` で解除できます。

## 自動起動（PC再起動後の使い方）

1. `a` キーで起動したいプロジェクトに `★` を設定する
2. 次回 `lazyprj` 起動時、tmuxセッションがゼロであれば `★` のプロジェクトを自動で一括起動
3. TUIのプロジェクト一覧から `Enter` でアタッチ先を選択

## キーバインド

`?` キーでヘルプオーバーレイを表示できます（全キーバインド一覧）。

### メイン画面

| キー | 動作 |
|------|------|
| `j` / `↓` | カーソル下移動 |
| `k` / `↑` | カーソル上移動 |
| `g` | 先頭へ移動 |
| `G` | 末尾へ移動 |
| `1` | Projects パネルにフォーカス |
| `2` | Detail パネルにフォーカス |
| `Tab` | Projects / Detail パネルのフォーカス切替 |
| `Enter` / `s` | セッション起動、または起動中ならアタッチ |
| `d` / `x` | セッション停止 |
| `R` | セッション再起動（確認ダイアログあり） |
| `e` | エディタ画面を開く（オーバーレイ表示） |
| `a` | 自動起動フラグのトグル（`★` 表示、即保存） |
| `A` | `★` のついた全プロジェクトを一括起動 |
| `X` | プロジェクトをスキップ済みへ移動 |
| `n` | スキャン画面を開く |
| `S` | スキャンディレクトリ設定画面を開く |
| `r` | プロジェクト一覧・セッション状態を再読み込み |
| `?` | ヘルプオーバーレイを表示 |
| `q` / `Ctrl+C` | 終了 |

### エディタ画面（`e` キーで起動）

メイン画面の上にオーバーレイ表示されます。ウィンドウとペインの追加・編集・削除ができます。

| キー | 動作 |
|------|------|
| `j` / `k` | ウィンドウ／ペイン移動 |
| `Tab` | Windows / Panes パネル切替 |
| `a` | ウィンドウ／ペインを追加 |
| `e` | 選択中の項目を編集 |
| `d` | 選択中の項目を削除 |
| `w` | 設定を保存 |
| `Esc` | メイン画面に戻る |

#### ペイン編集フォーム

| キー | 動作 |
|------|------|
| `j` / `k` | Dirリスト内を移動 |
| `e` | Dirを直接テキスト入力 |
| `Tab` / `Shift+Tab` | フィールド切替（Dir → Command → Execute） |
| `Space` | Execute フラグのON/OFF切替（Executeフォーカス時） |
| `Enter` / `w` | 保存 |
| `Esc` | キャンセル |

#### ウィンドウ編集フォーム

| キー | 動作 |
|------|------|
| `Tab` / `Shift+Tab` | フィールド切替（名前 → レイアウト） |
| `←` / `→` | レイアウト選択（レイアウトフォーカス時） |
| `Enter` / `w` | 保存 |
| `Esc` | キャンセル |

### スキャン画面（`n` キーで起動）

| キー | 動作 |
|------|------|
| `j` / `k` | カーソル移動 |
| `Space` | 選択トグル |
| `a` | 全選択 / 全解除 |
| `x` | スキップ済みに追加（スキップ済みアイテム上では解除） |
| `Space` / `Enter`（ヘッダー行） | スキップ済みセクションの展開 / 折りたたみ |
| `Enter` / `s` | 選択プロジェクトを追加して保存 |
| `Esc` / `q` | メイン画面に戻る |

スキップしたプロジェクトは `▶ スキップ済み (N)` セクションに格納され、設定ファイルの `skipped_paths` に記録されます。次回スキャン時に自動的に除外されます。

## tmuxレイアウト

エディタ画面でウィンドウのレイアウトを選択できます（プレビュー付き）。

| レイアウト | 説明 |
|-----------|------|
| `even-horizontal` | ペインを左右に均等分割 |
| `even-vertical` | ペインを上下に均等分割 |
| `main-horizontal` | 上に広いメイン、下に小ペイン |
| `main-vertical` | 左に広いメイン、右に小ペイン |
| `tiled` | グリッド状に配置 |

## アーキテクチャ

```
lazyprj/
├── main.go                    # エントリーポイント
├── go.mod
└── internal/
    ├── config/
    │   ├── config.go          # 設定の読み込み・保存・データ構造
    │   └── scan.go            # プロジェクトスキャン
    ├── tmux/
    │   └── tmux.go            # tmuxセッション操作（作成・停止・アタッチ）
    └── ui/
        ├── app.go             # メイン画面（Bubble Tea Model）
        ├── editor.go          # エディタ画面（ウィンドウ・ペイン編集）
        ├── scan.go            # スキャン画面
        ├── setup.go           # スキャンディレクトリ設定画面
        └── styles.go          # lipglossスタイル定義
```
